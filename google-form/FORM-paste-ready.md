Send ~5 days before. Takes ~2 minutes. Lets us tailor the room and gives everyone a warm start on the brand build.

## Setup guide (2 minutes)

1. Go to **forms.google.com → Blank form**. Paste the title + description above.
2. Add each question with the **type noted in bold** (Short answer / Paragraph / Multiple choice / Checkboxes). Toggle **Required** where marked.
3. Top-right **Settings → "Make this a quiz"** stays OFF. Under **Responses**, turn on **"Get email notifications for new responses"** if you want pings.
4. Hit **Send** → copy the link (use the shortened link) and share via WhatsApp/email.
5. **The night before:** open the **Responses → Summary** tab and skim. Pick **2–3 real answers** from Q5/Q6/Q7 to use as live examples in the deck (especially the Reveal and the Build). It makes the room feel personally seen.

## Why these questions (so you can defend/adjust them)
- **Q2 + Q3** set the level — if most are beginners, slow the prompting segment; if advanced, spend longer on the build.
- **Q4** decides phone-vs-laptop tool choices (e.g. lean on Gemini/Canva mobile if phones dominate).
- **Q6 (brand seed)** means nobody starts the Step-1 build from a blank page.
- **Q7** gives you a quick read on the room's aesthetic spread for the visual-identity step.
- **Q8** surfaces fears/ethics to pre-empt — feeds the Responsible-Use close.
- **Q9** = consent for the gallery walk, sorted upfront.
