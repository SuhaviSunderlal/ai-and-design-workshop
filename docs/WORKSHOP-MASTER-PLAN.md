# AI & Design — Workshop Master Plan

**Facilitators:** Suhav & Aditya
**Audience:** ~15 intelligent, educated adults (public-health + art backgrounds), mostly on laptops
**Duration:** 2.5 hours · 9:30–12:00 · in-person
**Tooling philosophy:** Free-tools-first. Paid options shown only as a clearly-labelled "Pro Path" sidebar so nobody is confused or feels excluded.
**Mode:** Balanced — teach a concept, then immediately build with it.
**Promised takeaway:** Every attendee leaves with their **own personal brand** captured in a reusable "Brand-in-a-Box" they can hand to *any* AI tool forever.

---

## The Creative Through-Line (Theme)

> **You are the Creative Director. AI is your genius intern.**

A genius intern is brilliant, fast, absurdly well-read, and eager — but *junior*. It does its best work when you:
- **Brief it well** → *prompting*
- **Don't overload its desk** → *context window / context rot*
- **Hand it references / a moodboard** → *reference images, examples*
- **Give feedback instead of expecting one-shot perfection** → *iteration*
- **Give it a permanent style guide so it never forgets who you are** → *the Brand-in-a-Box*

The intern is never the boss. You direct; it executes. This single metaphor carries every technical concept in the workshop without ever feeling juvenile — it's how real studios actually work.

**One-liner we repeat all morning:** *"AI isn't dumb. It's a genius intern waiting for a better brief."*

## Hands-On Builds

1. **Build #1 (Segment 2):** Rewrite a weak prompt into a Creative-Director brief using the prompt anatomy. Low stakes, fast win.
2. **THE BUILD (Segment 5) — Brand-in-a-Box:**
   - **Strategy:** Let AI *interview you*, then draft positioning, personality/archetype, mission, tagline.
   - **Visual identity:** Generate a color palette + font pairing + a logo concept on a free tool.
   - **Voice:** Define 3 voice traits + do/don't words.
   - **Apply:** Produce one real artifact — a branded social post or a one-panel brochure.
   - **Save:** Paste the brand into a reusable "brand block" / custom instructions / Project so every future chat already knows who they are.

## Easter Eggs (woven in, never announced)

- **The Reveal:** the cold-open image hides a small detail tied to the audience (e.g., a stethoscope + paintbrush) — revealed at the close.
- **The Broken Slide:** a deliberately "confused" slide during Segment 3 to *show* context rot, then fixed live.
- **Konami code** in the HTML deck unlocks a hidden "intern's confession" slide.
- **Rotating footer quips:** witty "intern" status lines at the bottom of slides.
- **A press-key secret slide** with a bonus prompt cheat.
